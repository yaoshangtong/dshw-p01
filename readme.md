## 二、数据来源
1. 个股行情：akshare，日度后复权行情，时间2020-01-01至今
2. 市场指数：沪深300(000300)作为市场基准，创业板指作为对比指数
3. 宏观数据：CPI同比增速（必选）、美元兑人民币汇率（自选），汇率可反映外资流向对A股影响
4. 财务数据：近5年净资产收益率ROE，整理为长格式数据表

## 三、存储方案
1. 基础存储：CSV格式，易读取、通用性强，适合日常数据分析；大批量数据下读写较慢、压缩率低
2. 进阶存储：Parquet列式存储，选择理由：金融时序数据字段多、体量偏大，Parquet压缩体积更小、读取速度更快，支持指定列读取
3. 日志记录：download_log.txt统一记录每一项数据下载状态

## 四、项目运行方式
1. 环境配置：pip install -r requirements.txt
2. 执行顺序
    1. 运行01_download.ipynb批量下载全部真实数据
    2. 运行02_clean.ipynb完成清洗、长宽表转换、格式对比
    3. 运行03_analysis.ipynb完成可视化与CAPM建模
3. 查看报告：打开根目录report.html即可阅读完整分析报告

## 六、分析结论简述
1. 股票收益率普遍呈现尖峰厚尾，不符合标准正态分布
2. 同行业标的收益率相关性更高，跨行业配置可有效分散风险
3. Beta大于1的个股多为周期成长行业，市场敏感度更高；Beta小于1为防御型行业
4. Alpha显著不为零代表个股存在独立于大盘的超额收益能力
